# SpriteBatchWindow Bug 修正實作計畫

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修正 code review 發現的 8 項錯誤（domain reload 靜默遺失資料夾、預覽紋理殘留、後綴重複、GUIStyle 共享狀態污染、HashSet 順序不確定、MaxTextureSize 損壞）。

**Tech Stack:** Unity 2021.3+, IMGUI (EditorGUI / GUILayout), EditorApplication.delayCall, ScriptableSingleton

### Task 4: Domain Reload 修正 — LoadState 拆分 + MaxTextureSize 驗證（Bug 1, 2 & 5）

**Files:**

- Modify: `Packages/com.wenrong.spritebatchslicer/Editor/SpriteBatchWindow.cs:43-53,275-292`

這是最重要的修正：AssetDatabase 在 domain reload 期間的 `OnEnable` 中尚未就緒，必須延遲載入資料夾 asset。同時，MaxTextureSize 的驗證也在這裡一併處理。

---

- [ ] **Step 1: 修改 `OnEnable` 與 `OnDisable`**

找到現有的 `OnEnable` 和 `OnDisable`：

```csharp
        private void OnEnable()
        {
            LoadState();
            InitFolderList();
            InitRectList();
        }

        private void OnDisable()
        {
            SaveState();
        }
```

替換為：

```csharp
        private void OnEnable()
        {
            LoadScalarSettings();
            InitFolderList();
            InitRectList();
            EditorApplication.delayCall += DelayedLoadFolders;
        }

        private void OnDisable()
        {
            EditorApplication.delayCall -= DelayedLoadFolders;
            SaveState();
        }
```

- [ ] **Step 2: 以 `LoadScalarSettings` + `DelayedLoadFolders` 取代現有的 `LoadState`**

找到現有的整個 `LoadState` 方法：

```csharp
        private void LoadState()
        {
            var s = SpriteBatchWindowState.instance;
            _settings.MaxTextureSize = s.MaxTextureSize;
            _settings.FilterMode = s.FilterMode;
            _settings.AlphaIsTransparency = s.AlphaIsTransparency;
            _settings.Compression = s.Compression;
            _settings.SpriteRects = new List<SpriteRectDef>(s.SpriteRects);
            _settings.TargetFolders.Clear();
            foreach (var path in s.FolderPaths)
            {
                var asset = AssetDatabase.LoadAssetAtPath<DefaultAsset>(path);
                if (asset != null)
                {
                    _settings.TargetFolders.Add(asset);
                }
            }
        }
```

整段替換為：

```csharp
        private void LoadScalarSettings()
        {
            var s = SpriteBatchWindowState.instance;
            _settings.MaxTextureSize      = System.Array.IndexOf(MaxTextureSizeValues, s.MaxTextureSize) >= 0
                                             ? s.MaxTextureSize
                                             : 2048;
            _settings.FilterMode          = s.FilterMode;
            _settings.AlphaIsTransparency = s.AlphaIsTransparency;
            _settings.Compression         = s.Compression;
            _settings.SpriteRects         = new List<SpriteRectDef>(s.SpriteRects);
        }

        private void DelayedLoadFolders()
        {
            if (this == null) return;
            var s = SpriteBatchWindowState.instance;
            _settings.TargetFolders.Clear();
            foreach (var path in s.FolderPaths)
            {
                var asset = AssetDatabase.LoadAssetAtPath<DefaultAsset>(path);
                if (asset != null)
                    _settings.TargetFolders.Add(asset);
            }
            RefreshTexturePaths();
            Repaint();
        }
```
